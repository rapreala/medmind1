# Adherence Feature Tests

This directory contains tests for the adherence tracking feature.

## Structure

- `data/` - Tests for data sources, models, and repository implementations
- `domain/` - Tests for entities, repositories (interfaces), and use cases
- `presentation/` - Tests for BLoCs, pages, and widgets

## Test Coverage

Tests should cover:
- Adherence log creation (taken, missed, snoozed)
- Adherence statistics calculation
- Real-time data streaming
- History retrieval and ordering
- Data export functionality
