# Profile Feature Tests

This directory contains tests for the user profile management feature.

## Structure

- `data/` - Tests for data sources, models, and repository implementations
- `domain/` - Tests for entities, repositories (interfaces), and use cases
- `presentation/` - Tests for BLoCs, pages, and widgets

## Test Coverage

Tests should cover:
- User profile display and updates
- Display name changes
- Notification preference management
- Theme preference persistence
- Profile photo upload
- Logout functionality
